Schedule tab — appointment booking

How users find shops and book appointments

What's in the Schedule tab
Upcoming appointments
List of scheduled appointments with shop name, service type, and date. Can cancel or mark as completed.
Find a shop
Search nearby mechanics and dealerships using Apple Maps. Shows name, distance, and rating. Tapping opens booking options.
Booking flow — how deep-linking works
  1. User taps a shop from search results
  2. App shows shop detail — name, address, phone, website
  3. "Book appointment" button opens shop's website in an in-app browser (Safari WebView)
  4. User completes booking on shop's own site
  5. Back in TuneUp, user saves a reminder: service type + date → creates an appointment record
  6. Maintenance reminder notification pauses until that date passes
Note: TuneUp doesn't integrate directly with shop booking APIs — too fragmented across thousands of shops. Instead it deep-links to their site and lets the user save a local reminder. Simple and works for any shop.
Notification silencing when appointment is scheduled
Once an appointment is saved for a given service type, the daily reminder notification for that service is suppressed. If the appointment date passes without a log entry being created, reminders resume the next day.
