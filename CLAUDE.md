## Project

**TuneUp** — a mobile app (iOS-first) for tracking vehicle maintenance history. Core differentiators:
- Location-based detection: notifies user when they've been at an auto shop/dealership for ~5 minutes, prompting them to log work
- Maintenance reminder engine: calculates upcoming due dates (oil changes, inspections, etc.) based on vehicle data and mileage, sends persistent notifications until the user logs the work or schedules an appointment
- Appointment scheduling: deep-links to mechanic/dealership booking pages
